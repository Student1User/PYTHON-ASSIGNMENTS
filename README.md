## 🚀 How to Contribute
1. **Fork** the repository.
2. **Clone** your fork:
   ```bash
   git clone https://github.com/your-username/PYTHON-ASSIGNMENTS.git
   ```
3. **Make changes** and commit:
   ```bash
   git add .
   git commit -m "Added new assignment"
   ```
4. **Push** the changes:
   ```bash
   git push origin main
   ```
5. Create a **Pull Request** (PR) for review.
